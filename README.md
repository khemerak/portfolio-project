# Khemerak Pav's Portfolio

This is the source code for my personal portfolio website, built with Next.js and Tailwind CSS. The portfolio showcases my skills, projects, and experience as a Computer Science graduate and IT professional.

## Features

- **Single-page application:** A smooth, single-page layout with scrolling navigation.
- **Dark/Light Mode:** A theme toggle to switch between dark and light modes.
- **Dynamic Content:** Project, skills, and education information is dynamically rendered from data within the components.
- **Interactive Background:** An interactive dot pattern background that responds to mouse movement.

## Technologies Used

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [magicui.design](https://magicui.design/)
- **Deployment:** Vercel (or your preferred platform)

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in your browser:**
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

You can easily customize the content of the portfolio by editing the data within the components located in the `src/components` directory.

-   **`Hero.tsx`**: Update your name, tagline, and contact information.
-   **`About.tsx`**: Update your personal summary.
-   **`Projects.tsx`**: Add, remove, or edit your projects.
-   **`Skills.tsx`**: Update your skills.
-   **`Education.tsx`**: Update your education and certifications.
